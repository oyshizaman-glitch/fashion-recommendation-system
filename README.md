# Fashion & Outfit Recommendation System

## Project Overview

The Fashion & Outfit Recommendation System is a database-driven e-commerce platform designed to simplify online fashion retail management. The system allows users to browse outfits, manage wishlists, place orders, provide ratings, and complete payments through multiple payment methods.

The project focuses on creating a centralized and scalable solution for both customers and administrators while maintaining accurate inventory management and secure transaction processing.



## Features

### Customer Features

* User Registration and Login
* Browse Fashion Products
* Add Products to Wishlist
* Place Orders
* Secure Payment Processing
* Product Rating and Reviews
* Outfit Recommendation Requests

### Administrative Features

* Product Management
* Inventory Tracking
* Warehouse Management
* Order Management
* Payment Monitoring
* Customer Data Management

### Inventory Management

* Warehouse-based Stock Tracking
* Size-based Inventory Control
* Real-time Quantity Updates
* Automatic Stock Deduction After Orders
